üéõÔ∏è Context-Sensitive Operation Modes

1. FACE SELECTION MODE (Extrusion Operations)
  • Extrusion Slider - Visual slider with precise numeric input
  • Direction Controls - Inward/outward toggle buttons
  • Taper Controls - Angled extrusions for complex joinery
  • Material Presets - 1/2" plywood, 3/4" hardwood, etc.
  • Real-time Preview - Live 3D preview before committing
2. OBJECT SELECTION MODE (Transform Operations)
  • Move Controls - X, Y, Z sliders with numeric inputs
  • Rotate Controls - X, Y, Z rotation wheels with degree inputs
  • Scale Controls - Uniform or per-axis scaling
  • Quick Rotation - 90¬∞, 180¬∞, 270¬∞ preset buttons
  • Center/Align Tools - Snap to grid, other objects
3. EDGE SELECTION MODE (Router Operations)
  • Router Bit Templates - Chamfer, roundover, ogee, cove, etc.
  • Depth/Radius Controls - Precise measurements for each operation
  • Profile Preview - 2D cross-section view of the cut
  • Direction Controls - Which side of edge to route
  • Multiple Edge Selection - Apply same operation to multiple edges
4. MULTI-SELECTION MODE (Array & Alignment)
  • Array Tools - Linear arrays, circular arrays, grid patterns
  • Alignment Tools - Align to face, edge, center points
  • Distribution Tools - Even spacing, specific intervals
  • Group/Ungroup - Manage object relationships
  • Batch Operations - Apply same operation to all selected
5. SKETCH MODE (2D Operations)
  • Dimension Controls - Length, width, radius adjustments
  • Constraint Tools - Parallel, perpendicular, tangent relationships
  • Fillet/Chamfer - Corner operations for 2D sketches
  • Pattern Tools - Rectangular, circular patterns
  • Offset Tools - Create offset curves for machining paths

üîß Technical Implementation Strategy

Frontend Architecture

  • HTML/CSS Panel - Overlay on 3D canvas with z-index management
  • Event-Driven Updates - Real-time response to selection changes
  • Modular Design - Each mode as separate, swappable component
  • Responsive Layout - Adapts to different screen sizes

Backend Integration

  • Selection Manager - Tracks what's selected and triggers mode changes
  • Operation Engine - Handles geometric transformations
  • Preview System - Non-destructive preview before applying
  • Undo/Redo Stack - Full operation history management

Performance Optimization

  • Lazy Loading - Only load active mode components
  • Debounced Updates - Prevent excessive real-time calculations
  • Efficient Rendering - Minimal DOM manipulation
  • Memory Management - Proper cleanup of unused modes
