Git Command Cheatsheet ("gcodes")

Basics

git status → show changes
git add . → stage all changes
git commit -m "message" → save changes
git push → upload to remote
git pull → download from remote

Branching

git branch → list branches
git checkout branch → switch branch
git checkout -b new-branch → create & switch
git push -u origin new-branch → publish new branch

Fixing / Resetting

git reset --hard → nuke uncommitted changes
git reset --hard branch → match another branch
git checkout . → throw away file changes

Branch Management

git branch -d branch → delete branch (local)
git push origin --delete branch → delete branch (remote)

Shortcuts ("gcodes")

gswitch → quickly checkout branch
gnew → create & push new branch
rollback → reset to last commit

Useful

pwd → show current folder
ls → list files
