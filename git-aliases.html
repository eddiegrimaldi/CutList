Install all aliases (one-time)

# Safe fast-forward pull
git config --global alias.gup "pull --ff-only"
# Safer push (protects against blind overwrites)
git config --global alias.gpush "push --force-with-lease"
# Short status + branch tracking
git config --global alias.gstat "status -sb"
# Last commit (one line)
git config --global alias.glast "log -1 --oneline"
# Compact graph of history (all branches)
git config --global alias.glog "log --oneline --graph --decorate --all"
# Clean untracked files & folders (⚠️ destructive)
git config --global alias.gclean "clean -fdx"
# Undo last commit but keep changes staged
git config --global alias.gundo "reset --soft HEAD~1"

Verify any alias: git config --get alias.<name>

One-line installer (copy/paste)

git config --global alias.gup  "pull --ff-only" \
 && git config --global alias.gpush "push --force-with-lease" \
 && git config --global alias.gstat "status -sb" \
 && git config --global alias.glast "log -1 --oneline" \
 && git config --global alias.glog  "log --oneline --graph --decorate --all" \
 && git config --global alias.gclean "clean -fdx" \
 && git config --global alias.gundo "reset --soft HEAD~1"

Daily workflow (90% of life)

1) Update to latest

git gup
git glast   # sanity check what you just pulled

2) Work normally

# edit files ...
# stage + commit (example)
git add -A && git commit -m "Your useful message"
# optional: view short repo state
git gstat

3) Push safely

git gpush

Uses --force-with-lease so you don’t clobber work you haven’t seen.

4) Browse history (when confused)

git glog
